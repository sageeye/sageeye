How to Create Your own Landing

Step 1:

After downloading the react files you will get three folder.

We will need to work on two folders for our task. First, the main theme folder named custar-v1 to select and copy the section components from, and then, the custar-starter-template for creating a custom landing based on your selection of section components devoid of any extra code.

Step 2:

Now we need to open command line in the “custar-starter-template” directory and install yarn by running the command yarn in the cmd.

Step 3:

After installing Yarn in our custar-starter-template directory we have to choose the sections we want in our landing template.


To take our desired section component, we will have to go to the custar-v1/src/sections directory and simply copy the sections. Then we need navigate to custar-starter-template/src/sections directory and paste the sections there.

Step 4:

Now we need to open the command line in this directory and start the yarn by typing “yarn start” in the command line.

Note: You may notice some warning saying “could not find image” and [gatsby-plugin-image] Could not read image data because the image paths are mismatched. Donut worry! We will show you on step-8 how to fix that. Though it won’t cause any issue to starting our app.

Final Step: Generating production build

Running the command yarn build will generate production build of the page. You will see a folder named public in custar-starter-template’s root directory. Use that folder for production. You now have a unique custom landing of your choice. Enjoy!
